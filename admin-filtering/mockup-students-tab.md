# UI/UX Wireframe & Spec: Students Tab

## 1. Visual Layout
**Style:** Dark Mode interface. Textless logo in the header.

### Top Section: High-Level Roster Metrics
* **KPI Card 1:** Active Students (e.g., 142)
* **KPI Card 2:** Average Turn Count per Session (e.g., 6.4)
* **KPI Card 3:** Mode Preference (e.g., 68% Socratic / 32% Direct)

### Bottom Section: Anonymized Roster Table
* **Purpose:** Track individual engagement without violating FERPA.
* **Columns:**
  * **Anonymized ID:** (e.g., Student_B4M2)
  * **Total Questions Asked:** (e.g., 24)
  * **Favorite Mode:** Socratic
  * **"Cannot Find Info" Triggers:** 3
  * **Resolution Rate:** 85% successfully resolved
* **Row Interaction:** Clicking a student row applies a global filter to the "Chats" tab, showing only that student's historical transcripts.
